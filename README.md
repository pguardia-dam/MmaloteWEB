## 🧩 Característiques

- 🔥 Combat per torns  
- 🧙‍♂️ Sistema gestor d’habilitats    
- 🌐 Integració amb API pròpia  
- 💾 Sistema de login i dades de jugador

- **C#**
- **API REST (Node/Express / ASP.NET / el que facis servir)**
- **GitHub Pages**
- **Itch.io**
